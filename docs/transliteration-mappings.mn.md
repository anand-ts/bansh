# Галигийн хөрвүүлэлтийн хүснэгт

Англи хувилбар: [transliteration-mappings.md](./transliteration-mappings.md)

Энэ баримт бичиг нь одоогоор `Sources/Core/TransliterationSession.cpp` дотор тодорхойлсон дүрмүүдийг тусгасан болно.

## Тэмдэглэл

- Хүснэгтүүдийг уншихад хялбар байлгахын тулд оролтыг тогтсон том үсгийн хэлбэрээр үзүүлсэн. Код нь үсгийн том, жижиг хэлбэрийг ялгаж ажилладаг тул жижиг үсэг болон холимог том, жижиг хувилбарууд мөн ажиллана.
- Хөдөлгүүр хамгийн урт тааралтыг түрүүлж сонгодог. Жишээлбэл, `CH` нь `C`-ээс өмнө, `III` нь `II` эсвэл `I`-ээс өмнө таарна.
- Шинээр эхэлсэн хөрвүүлэлт бүр эр зохицлын төлөвөөс эхэлнэ.
- Зарим хөрвүүлэлт одоогийн зохицлын төлөвийг өөрчилнө:
  - эр зохицлын төлөвт шилжүүлэгч: `A`, `AI`, `YA`, `YO`, `YU`, мөн `"` тэмдэгтэй хүчээр сонгосон `O` / `U` хэлбэрүүд
  - эм зохицлын төлөвт шилжүүлэгч: `E`, `EI`, `Q`, `QI`, `W`, `WI`, `YE`, мөн `'` тэмдэгтэй хүчээр сонгосон `O` / `U` хэлбэрүүд
- `O` / `OI` болон `U` / `UI` гэсэн хоёрдмол эгшгийн дүрэм нь одоогийн зохицлын төлвөөс хамаарна.
- Хэрэгжүүлэлтийн төгсгөл хэсгийн холимог бичгийн дүрмүүд нь хэрэглэгч идэвхтэй хөрвүүлэлтийн дотор засвар хийх үед аль хэдийн кирилл болсон текстийг зөв үргэлжлүүлэн хөрвүүлэхийн тулд орсон.
